git addмқҳ лң»

  • к№ғмқҳ лҸҷмһ‘мқ„ мқҙн•ҙн–җл Өл©ҙ лЁјм Җ мӣҢнӮ№ л””л үн„°лҰ¬ лҳҗлҠ” нҠёлҰ¬лқјкі  н•ңлӢӨ.
  • мҡ°м„  л‘җк°Җм§Җл§Ң кё°м–өн•ҳл©ҙ лҗңлӢӨ.
    • untracked мғҒнғң: мӢӨм ң мһ‘м—…нҢҢмқј л°Ҹ нҸҙлҚ”к°Җ мһҲлҠ”лҚ°, мқҙ кіөк°„мқ„ мһҗлҸҷмңјлЎң м¶”м Ғн•ҳм§Җ м•ҠлҠ”лӢӨ. мҰү, мғҲлЎң л§Ңл“  нҢҢмқјмқҖ лӘЁл‘җ untracked мғҒнғңмқё кІғмқҙлӢӨ.
    • tracked мғҒнғң: Gitмқҙ м¶”м Ғмқ„ н•  мҲҳ мһҲлҸ„лЎқ git add лӘ…л №м–ҙлҘј мӮ¬мҡ©н•ңлӢӨ.
  • к№ғмқҖ мҡ”мІӯл°ӣмқҖ нҢҢмқјл“Өл§Ң м¶”м Ғ кҙҖлҰ¬н•ңлӢӨ.

git addмқҳ лң», stage & unstage

  • м¶”м Ғ мғҒнғңк°Җ лҗң л’Өм—җлҠ” мһ„мӢң мҳҒм—ӯм—җ н•ҙлӢ№лҗҳлҠ” мҠӨн…Ңмқҙм§Җ кіөк°„мңјлЎң л“Өм–ҙк°„лӢӨ.
  • мҰү, git addк°Җ лҒқлӮҳл©ҙ м»Өл°Ӣмқ„ н•ҳкё° м „ лӢЁкі„мқё мһ„мӢң мҳҒм—ӯмңјлЎң л“Өм–ҙк°ҖкІҢ лҗңлӢӨ.
  • мқҙ л•Ң кө¬л¶„м җмқҖ stage мғҒнғңмҷҖ unstage мғҒнғңмҷҖ кө¬л¶„лҗңлӢӨ.
    • к·ёлҹ¬лӮҳ, мӢӨм§Ҳм ҒмңјлЎңлҠ” лҢҖк°ңмқҳ кІҪмҡ° tracked л¶Җл¶„кіј untracked л¶Җл¶„мқҳ мҳҒм—ӯмқҖ нҒ¬кІҢ м°Ёмқҙк°Җ м—ҶлӢӨ.
  • к·ёлҹ°лҚ°, мҳҲмҷёк°Җ мһҲлҠ”лҚ°, мҠӨн…Ңмқҙм§Җ мҳҒм—ӯм—җ мһҲлҠ” нҢҢмқјкіј мӣҢнӮ№ л””л үн„°лҰ¬ м•Ҳм—җ мһҲлҠ” нҢҢмқј лӮҙмҡ©м—җ м°Ёмқҙк°Җ мһҲмқ„ кІҪмҡ° unstage мғҒнғңк°Җ лҗңлӢӨ.
    • мҰү, нҢҢмқјмқҙ мҲҳм •н•  кІҪмҡ° мһ„мӢңм ҒмңјлЎң мҠӨн…Ңмқҙм§Җ лӘ©лЎқм—җм„ң м ңкұ°к°Җ лҗң кІғмқҙкё° л•Ңл¬ём—җ н•ҙлӢ№ нҢҢмқјл§Ң git addлЎң м¶”к°Җн•ҳл©ҙ лҗңлӢӨ.
  • мқҙ мҜӨм—җм„ң git лқјмқҙн”„ мӢёмқҙнҒҙмқ„ к·ёлҰјмңјлЎң нҷ•мқён•ҙліёлӢӨ.

git addмқҳ лң», modified & unmodified

  • мҪ”л“ңлҘј ліҖкІҪн•ңлӢӨлҠ” лң»мқҖ нҢҢмқјмқ„ мҲҳм •н•ңлӢӨлҠ” лң»мқҙлӢӨ.
  • к·ёлҹ°лҚ°, нҢҢмқјмқҙ мҲҳм •лҗҳл©ҙ, мҠӨн…Ңмқҙм§Җм—җм„ңлҠ” мҲҳм •н•ң нҢҢмқјкіј мӣҗліё нҢҢмқјмқ„ кө¬л¶„н•ҳкё° мң„н•ҙ мҲҳм •н•Ё(modified)мҷҖ мҲҳм •н•ҳм§Җ м•ҠмқҢ(unmodified) мғҒнғңлЎң н‘ңнҳ„н•ңлӢӨ.
  • modifiedмқҳ мқҳлҜёлҠ” к№ғмқҙ мӢӨм ңлЎң кё°лЎқн•ң нҢҢмқјмқҙл©°, мӢӨм§Ҳм ҒмңјлЎң лІ„м „мқ„ мқҳлҜён•ңлӢӨ.
    • мқҙ л•Ң, к№ғ commitмқ„ н•ҳл©ҙ мқҙ ліҖкІҪ лӮҙм—ӯмқҖ мҳҒкө¬м ҒмңјлЎң кё°лЎқлҗңлӢӨ.
    • л§Ңм•Ҫ, нҢҢмқј мҲҳм •мқ„ н•ҳкІҢ лҗҳл©ҙ мқҙлҠ” мһ¬ л“ұлЎқмқ„ мқҳлҜён•ңлӢӨ. л”°лқјм„ң мһ¬ л“ұлЎқмқ„ н•ҳл Өл©ҙ git addлҘј мһ‘м„ұн•ҙм•ј н•ңлӢӨ.
  • unmodifiedмқҳ лң»мқҖ мҲҳм •н•ҳм§Җ м•Ҡм•ҳмқҢмқ„ мқҳлҜён•ңлӢӨ.

мҡ”м•Ҫ

  • лӢӨм–‘н•ң мқҳкІ¬мқҙ мҳӨк°”кё° л•Ңл¬ём—җ н•ңлІҲ лҚ” м„ӨлӘ…н•ҳл©ҙ м•„лһҳмҷҖ к°ҷлӢӨ. мҰү, мҠӨн…Ңмқҙм§ҖлҘј кё°мӨҖмңјлЎң мӮјмһҗ.
