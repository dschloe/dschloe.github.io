I. SmartPLS 설치

  • SmartPLS는 구조방정식모델링을 위한 전용 통계분석 프로그램으로 다양한 학문 분야에서 광범위하게 사용됨.
  • 편이성, 친숙성, 안정성 면에서 매우 우수함.
  • 매우 적은 소표본에서도 활용할 수 있음.
  • 정규분포 등의 엄격한 가정 조건에 구애받지 않고 사용할 수 있음.
  • 프로그램 설치는 아래 링크를 클릭한다.
    • Download latest version - SmartPLS 3.3.2
  • 학생용 버전은 평생 무료이며, 약간의 사용상 제약이 존재한다.

II. SmartPLS 관련 자료

  • 교제 추천

    • 원서 - A Primer on Partial Least Squares Structural Equation Modeling(PLS-SEM) (2013)
    • 번역서 - PLS 구조모델의 이해 (2014)
    • 국내저서 - 석박사학위 및 학술논문 작성 중심의 SmartPLS 3.0 구조방정식모델링 (2018)

  • 온라인 강의도 존재하지만, 무료는 아니다.

    • 교제, 영상 및 기타 자료는 Get to Know PLS-SEM and SmartPLS에서 확인한다.

III. 데이터 준비와 저장

  • 설문조사가 완료된 데이터셋이 필요하다.
    • 데이터 다운로드
  • 데이터셋의 기본적 설명은 다음과 같다.
    • 잠재변수 5개 - 시스템품질, 정보품질, 서비스품질, 모바일러닝 만족도, 학습전이의도
    • 인구통계변수 1개 - 성별(Gender)
    • 척도화: 5점 리커트 척도
    • 총 사례 수: 89개

IV. SmartPLS 기본 운용법

  • 설치된 프로그램을 실행한 뒤, [새프로젝트 만들기]-[프로젝트명]을 입력한다.

  • [두 번 클릭하여 데이터를 불러오기!]를 클릭하여 다운로드 받은 데이터 파일을 불러온다.
    • 이 때 기초통계량이 자동 계산되어 보여진다.

  • R 프로그램에서 데이터를 불러와서 간단하게 확인해보자.
url <- "https://raw.githubusercontent.com/chloevan/datasets/master/survey/mobile_satisfaction.csv"
data <- read.csv(url, header = TRUE)
str(data)
